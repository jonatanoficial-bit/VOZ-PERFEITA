# Mitos comuns (e alternativas)

## “Apoio é empurrar abdômen”
Alternativa: controle de fluxo (sss) + SOVT + frase leve.

## “Para cantar alto tem que fazer força”
Alternativa: dinâmica controlada + ressonância + coordenação.

## “Rouquidão é normal”
Não. Rouquidão é sinal de carga/técnica inadequada ou fadiga.

## “Agudo bom é gritado”
Agudo bom é ajustado: menos volume, mais eficiência.
